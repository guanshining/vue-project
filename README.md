# vue-project
    1、git clone   https://github.com/guanshining/vue-project.git
    2、npm install
    3、npm run dev
    
    组件如何使用及所需参数在组件中有写
    
# 项目结构
    src 源码
        base       公共组件
        components 页面组件
        App.vue    主模板文件
        main.js    主入口文件
        router     路由
        
    public 公共资源
    
    package.json        所需模块
    webpack.config.js   打包配置文件
